耳鸣频率初步诊断模块
用户通过交互式音频测试（类似纯音测听简化版）自主匹配耳鸣主频（例如：500 Hz ~ 16 kHz 范围内滑动选择）。
支持保存多个耳鸣频率记录（如双耳不同频率）。
提供语音/文字引导，确保操作安全（避免高音量损伤听力）。
个性化治疗音乐生成模块
基于用户输入的耳鸣频率，自动生成 “个体化切迹音乐”（TMNMT）：
以耳鸣频率为中心，移除 1/2 八度音程 的频段（陷波滤波）。
支持从本地音乐库导入任意歌曲（MP3/WAV/AAC 等常见格式）。
实时处理音频，生成带“切迹”的治疗版本（可预览）。
同时支持 自然声音库（如雨声、溪流、白噪声等）作为备选治疗声。
治疗计划与时间管理模块
每次治疗会话 不得少于 30 分钟（强制设定，不可跳过）。
每日累计治疗时间 建议 ≥ 2 小时，≤ 6 小时：
自动记录每日播放时长（后台服务+前台通知）。
达到 2 小时时显示“今日目标达成”；超过 6 小时弹出提醒“避免过度使用”。
支持设置每日治疗时段（如早、中、晚三次，每次 40 分钟）。
音量智能控制模块
音量默认设为 略低于耳鸣感知响度（用户可手动微调）。
提供“舒适音量测试”引导（避免掩蔽过度）。
支持夜间模式（自动降低最大音量上限，保护睡眠）。
本地音乐集成模块
允许用户从手机本地选择音乐文件作为治疗素材。
显示专辑封面、歌曲名、时长。
对选中歌曲实时应用“切迹滤波”处理（需注明处理可能耗时几秒）。
数据记录与反馈模块
记录每日治疗时间、使用频率、主观感受评分（1~5 星）。
生成周报/月报（可导出 PDF 或分享给医生）。
提供“耳鸣困扰量表（THI 简化版）”定期评估。
